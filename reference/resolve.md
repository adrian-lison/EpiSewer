# Resolve a lazy EpiSewer object

Resolves a lazy `tbp`, `tbe`, or `tbc` object by evaluating it once all
required inputs are available.

## Usage

``` r
resolve(x, ...)
```

## Arguments

- x:

  A lazy object of class `tbp`, `tbe`, or `tbc`.

- ...:

  Further arguments passed to the method.

## Value

The resolved value or updated modeldata.
