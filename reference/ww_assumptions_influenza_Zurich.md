# Example sewer_assumptions object for Influenza A in Zurich

Epidemiological assumptions for Influenza A, including generation time
and shedding load distributions derived from the literature.

## Usage

``` r
ww_assumptions_influenza_Zurich
```

## Format

A `sewer_assumptions` object as returned by
[`sewer_assumptions()`](https://adrian-lison.github.io/EpiSewer/reference/sewer_assumptions.md).
