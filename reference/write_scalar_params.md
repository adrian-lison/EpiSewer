# Identify scalar parameters in a stan model and write them to a text file

Identify scalar parameters in a stan model and write them to a text file

## Usage

``` r
write_scalar_params(
  modelfolder = "inst/stan",
  modelname = "EpiSewer_main.stan"
)
```

## Arguments

- modelfolder:

  The folder containing the stan model file.

- modelname:

  The name of the stan model file.
