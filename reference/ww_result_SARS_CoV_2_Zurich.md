# Example EpiSewerJobResult for SARS-CoV-2 in Zurich

Result of running
[`EpiSewer()`](https://adrian-lison.github.io/EpiSewer/reference/EpiSewer.md)
on
[ww_data_SARS_CoV_2_Zurich](https://adrian-lison.github.io/EpiSewer/reference/ww_data_SARS_CoV_2_Zurich.md)
with
[ww_assumptions_SARS_CoV_2_Zurich](https://adrian-lison.github.io/EpiSewer/reference/ww_assumptions_SARS_CoV_2_Zurich.md).
Can be used to explore outputs and plotting functions without fitting a
model.

## Usage

``` r
ww_result_SARS_CoV_2_Zurich
```

## Format

An `EpiSewerJobResult` object as returned by
[`EpiSewer()`](https://adrian-lison.github.io/EpiSewer/reference/EpiSewer.md).
