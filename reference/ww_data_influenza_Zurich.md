# Example sewer_data object for Influenza A in Zurich

Processed wastewater data for Influenza A from the WWTP Werdhölzli in
Zürich (September–December 2022), ready for use with
[`EpiSewer()`](https://adrian-lison.github.io/EpiSewer/reference/EpiSewer.md).

## Usage

``` r
ww_data_influenza_Zurich
```

## Format

A `sewer_data` object as returned by
[`sewer_data()`](https://adrian-lison.github.io/EpiSewer/reference/sewer_data.md).
