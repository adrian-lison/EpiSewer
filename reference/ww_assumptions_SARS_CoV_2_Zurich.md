# Example sewer_assumptions object for SARS-CoV-2 in Zurich

Epidemiological assumptions for SARS-CoV-2, including generation time,
incubation period, and shedding load distributions derived from the
literature.

## Usage

``` r
ww_assumptions_SARS_CoV_2_Zurich
```

## Format

A `sewer_assumptions` object as returned by
[`sewer_assumptions()`](https://adrian-lison.github.io/EpiSewer/reference/sewer_assumptions.md).
